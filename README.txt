# Md Fayaz Ahamed's Personal Website

This repository contains the code for my personal website. The website showcases my academic background, research, skills, and experience in fields like Machine Learning, Generative AI, Data Science, and Mathematics.

## Features
- **Personal Introduction**: Highlights my qualifications, experience, and expertise.
- **Research & Projects**: Showcases key research and projects in Machine Learning and AI.
- **Contact Information**: Provides a way to get in touch for collaborations or job opportunities.

## Technologies Used
- **HTML5** for structure and content
- **CSS/SCSS** for styling and responsive design
- **JavaScript** for interactivity
- **FontAwesome** for icons
- **jQuery** and **Scrollex** for smooth scrolling effects
